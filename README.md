# Open Outpost
> You have failed. Our colony is doomed.

When I was a teenager, I was very into a game called [Outpost 2: Divided Destiny](https://en.wikipedia.org/wiki/Outpost_2:_Divided_Destiny).
This is my attempt to create a game in a similar vein while learning [Phaser JS](https://phaser.io/).
Like nearly every project in my life, I will likely abandon it.

You can demo my laughable progress here: [https://mrkrstphr.github.io/open-outpost]
